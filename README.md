# Rocketseat-NLW_22_Operator-ClipMaker_AI
Projeto da trilha iniciante do evento da Rocketseat NLW-22 (2026), onde foi proposto criar uma página de corte de vídeo com o auxilio da IA. A ideia é que ao subir um vídeo, a IA analise e sugere um trecho que seja viral, ela faz o corte e retorna este corte na tela.
